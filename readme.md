# Installation

* create and activate environment using [vox](https://xon.sh/python_virtual_environments.html) and [virtualenv](https://virtualenv.pypa.io/en/latest/)
```
virtualenv env
vox activate env
```

* install dependencies
```
pip install -r requirements.txt
```
